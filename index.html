I Want to Learn to Code for Good.

A few years ago, after working as a project manager in technology for many years, I decided to take time off. Shortly after, Occupy Wall Street happened and I became involved in the technology working group. We were working on developing technology solutions for the movement. I started to understand the importance of activist movements having tools that are owned and controlled by them, not private companies. I became interested in the idea of building free software for the purpose of helping movements working for positive change to organize and communicate.

What I'm looking to do is develop the skills needed to be able to help build such tools. Specifically, I'm want to learn to use JavaScript and JQuery (and helpful frameworks like Backbone, Angular, React, etc) to build web applications that bring together tools and information from disparate places to help organize and engage.
